Rider
=====
Python Web Application Framework - powered by Falcon (http://falconframework.org/), inspired by Django

Rider provides the system with flexibility to create complex projects.

Install
-------

```bash
$ pip install rider==0.1.0a
```

Vision
------
* be thin
* be simple
* be clean
* be elegant

* try consistent
* try to be fast
* try not to be magic
