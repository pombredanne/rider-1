=====
Python Web Application Framework - powered by Falcon (http://falconframework.org/), inspired by Django

Install
-------

```bash
$ pip install rider==0.1.4a
```

Vision
------
* be thin
* be simple
* be clean
* be elegant

* try consistent
* try to be fast
* try not to be magic
